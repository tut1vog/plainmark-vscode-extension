# HTMLTag inline — typographic tags in paragraphs (CommonMark §6.6)

Keyboard shortcuts: press <kbd>Ctrl+C</kbd> to copy and <kbd>Ctrl+V</kbd> to paste.

Math notation: water is H<sub>2</sub>O and exponent x<sup>2</sup> behaves.

Highlighting: this sentence has <mark>highlighted text</mark> mid-prose.

Abbreviations: see <abbr title="HyperText Markup Language">HTML</abbr> spec.

Quote: <q>short inline quote</q> with closing tag.

Tags inside emphasis: this is *italic with <kbd>tag</kbd> inside* nested.

Tags adjacent: <kbd>A</kbd><kbd>B</kbd><kbd>C</kbd> with no gap.

Tag with attribute: <span class="foo" data-x="y">styled span</span> with attributes.
