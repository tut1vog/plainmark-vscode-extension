# HTMLTag inline — self-closing and void elements

XHTML-style self-closing: line one<br/>line two<br/>line three.

HTML5-style void elements: line one<br>line two<br>line three.

Self-closing with attributes: <img src="x.png" alt="x"/> mid-prose.

Mixed: text <hr/> then more text (note: an inline `<hr/>` is HTMLTag, not the markdown-HR block).

Self-closing followed by closing punctuation: end of paragraph<br/>.
