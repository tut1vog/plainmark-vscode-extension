Place caret at the end of the empty `> ` line (after the trailing space, which should be invisible in the rendered chrome). Press Enter once. The empty `> ` should strip in a single keystroke and the caret should land on a new plain line below — empty-quote-line exit regression for callouts. Works at any depth (the keymap doesn't care that this is a callout).
