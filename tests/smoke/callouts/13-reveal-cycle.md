> [!NOTE] Title to edit
> Body line for caret parking.

Use this file for the reveal-on-caret-in-header interaction:
1. Park caret on the body line. Header shows icon + "Title to edit" + no raw bytes.
2. Click into the header line. Widget collapses; raw `[!NOTE] Title to edit` reappears for editing.
3. Edit the type to `[!WARNING]` — still revealed.
4. Arrow off the line. Widget re-renders as a `Warning` callout with new accent + icon.
