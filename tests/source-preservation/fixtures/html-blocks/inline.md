# Inline raw HTML fixture

A paragraph with <kbd>Ctrl+C</kbd> and <kbd>Ctrl+V</kbd> for clipboard shortcuts.

Superscript <sup>2</sup> and subscript <sub>n</sub> mid-prose.

Self-closing void elements: line one<br/>line two<br/>line three.

Mid-paragraph <!-- comment --> stays inline.

Mixed line: text <mark>highlighted</mark> with <span style="color:red">styled span</span> and a `<code>literal</code>` inside backticks (the backticks make it a markdown code span, not an HTMLTag).
